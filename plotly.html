Plotly

This page is for Plotly.

library(tidyverse)
library(p8105.datasets)

library(plotly)

Here, I chose the instacart data.

instacart = 
  instacart %>% 
  select(order_id, product_id, order_number, order_dow, order_hour_of_day, product_name, aisle, department) %>% 
  drop_na()

First, we can check the number of orders for different deparment during the week.

instacart %>%
  count(order_dow, department) %>%
  plot_ly(
    x = ~order_dow, y = ~n, type = "scatter", mode = "lines", color = ~department, colors = "viridis"
  )

Next, we can check the boxplot of the distributions of order_number during a day.

instacart %>% 
  sample_n(4000) %>% 
  plot_ly(
    x = ~order_hour_of_day, y = ~order_number, type = "box", colors = "viridis"
  )

At last, what aisles are popular? let’s compare number of orders from different aisles by making a bar plot.

instacart %>% 
  group_by(aisle) %>% 
  summarise(n_obs = n()) %>% 
  filter(n_obs > 20000) %>% 
  arrange(n_obs) %>% 
  plot_ly(
    x = ~reorder(aisle, -n_obs), y = ~n_obs, type = "bar", colors = "viridis"
  )
